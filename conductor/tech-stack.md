# Tech Stack: Maternity Pay Calculator

## Frontend
- **React (TypeScript)**: Used for building the interactive dashboard with strong type safety and component-based architecture.
- **PWA**: Configured for offline access, fast loading, and installability.

## Styling
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development and consistent design.

## Data Visualization
- **Victory / ApexCharts**: Chosen for creating the interactive payment timeline graphs.

## State Management
- **Zustand / Jotai**: Lightweight and efficient state management for handling user inputs, policy rules, and calculation results.

## Build and Tooling
- **Vite**: For fast development and optimized production builds.
- **Vitest / React Testing Library**: For comprehensive unit and integration testing.
